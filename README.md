# Практика "Игра" | Дизайн-документ
## Основное
- Платформа: `ПК`
- Технологии: `C#` `MonoGame`
- Язык: `Русский`
- Жанры: `2D` `Top-Down`
- Настроение: `Мрачное`
- Сеттинг: `Постапокалиптика`

## Сюжет
Главный герой - Вояка, которому надо избавить мир от захвативших его киборгов.

## Игровой мир
Один большой открытый мир с главной целью в центре карты. Чем ближе к цели, тем сильнее противники.

### Возможные варианты биомов:
- `Выжженная земля`
- `Городские развалины`

## Геймплей
Карта одна. В зависимости от выбранного персонажа спавн будет происходить в разных локациях с разным оружием.
Надо будет добраться в определенную точку, попутно зачищая карту от киборгов.
Концовка также будет отличаться за разных персонажей

### Оружие
Виды оружия: `Пистолет` `Автомат` `Дробовик`

Характеристики оружия:
- `Урон`
- `Скорость атаки`
- `Площадь атаки`
